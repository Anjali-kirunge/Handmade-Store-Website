# 🛍️ Handmade Store UI

A modern and responsive **Handmade Store website UI** built using HTML, CSS, and Bootstrap. This project showcases a clean e-commerce front-end design with product listings, categories, and cart layout.

## 🚀 Features
- Responsive navigation bar
- Image carousel (Hero section)
- Shop by category section
- Product listing cards
- Cart summary UI 🛒
- Contact form
- Footer with useful links
- Fully responsive using Bootstrap

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap 5

## 📂 Project Structure
- `index.html` – Main UI layout
- `style.css` – Custom styling (if used)
- External images via URLs

## 🎯 Key Highlights
- Clean and modern e-commerce UI design
- Bootstrap components like carousel, cards, and grid system
- Beginner-friendly project for front-end development
- No backend (UI-only project)

## 📸 Sections Included
- Navbar
- Hero Carousel
- Categories
- Products Grid
- Cart Summary
- Contact Form
- Footer

## 📌 How to Run
1. Clone or download the repository
2. Open `index.html` in your browser
